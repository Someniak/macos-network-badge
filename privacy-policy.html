Network Badge — Privacy Policy

Last updated: March 21, 2026

Overview

Network Badge is a network quality monitor designed for travelers. It records GPS-tagged network latency measurements so you can see where your internet connection was good or bad on a map.

All data stays on your device. Network Badge has no servers, no cloud sync, no analytics, and no advertising.

Data We Collect

Network Badge records the following data locally on your device:

  • GPS location — Your precise coordinates at each measurement point. Used to map network quality along your route.
  • Network quality metrics — Latency (ping time), connection type (WiFi, Cellular, etc.), success/failure of each measurement.
  • WiFi network name (SSID) — The name of the WiFi network you're connected to, so you can identify which networks had good or poor quality.
  • WiFi signal strength (RSSI) — Signal strength in dBm, when available.
  • Speed and altitude — Estimated travel speed and GPS altitude, used for quality analysis and map rendering.

How Data Is Stored

  • All data is stored in a local SQLite database on your device.
  • On iOS: Application Support/NetworkBadge/quality.db
  • On macOS: ~/.networkbadge/quality.db
  • Data is append-only by design — no automatic deletion.

Data Sharing

We never share your data with anyone. Specifically:

  • No data is sent to our servers (we don't have servers).
  • No data is shared with third parties.
  • No advertising networks or analytics SDKs are included.
  • No user tracking or profiling of any kind.

The only network requests the app makes are:

  • HTTP latency measurements — A small GET request to Apple's captive portal server (captive.apple.com) or a user-configured URL, to measure network latency. No personal data is sent.
  • IP geolocation (optional, off by default) — A request to a public geolocation API to estimate your location when GPS is unavailable. Only your IP address is sent (inherent to the request).
  • Update checks — Periodic checks to GitHub Releases to see if a new version is available. No personal data is sent.

Location Data

Network Badge requests location permission to map network quality along your route. On iOS, the app may request "Always" location permission to continue recording in the background while you travel. You can change this at any time in Settings → Privacy → Location Services.

Location data is stored only on your device and is never transmitted.

Data Export and Deletion

  • Export: You can export your data as a CSV file at any time from the History tab.
  • Delete: You can delete all data by removing the app. On macOS, delete the ~/.networkbadge/ directory.

Children's Privacy

Network Badge does not knowingly collect data from children under 13. The app does not require an account and does not collect personal information.

Changes to This Policy
